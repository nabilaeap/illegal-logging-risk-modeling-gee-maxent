# QGIS Spatial Interpretation Workflow

## Objective
Identify environmental characteristics at illegal logging locations.

## Steps
1. Load MaxEnt suitability map.
2. Load environmental raster variables.
3. Overlay illegal logging occurrence points.
4. Use Identify Features tool.
5. Click location point to extract:
   - elevation
   - slope
   - distance to river
   - distance to road
   - ndvi
   - landcover
   - population density

## Output Interpretation
Each logging location can be analyzed based on surrounding environmental conditions.
