# Illegal Logging Risk Modeling using Google Earth Engine and MaxEnt

## Overview
This project analyzes potential illegal logging risk areas using spatial environmental variables processed in Google Earth Engine and modeled using the Maximum Entropy (MaxEnt) approach. The objective is to identify environmentally suitable areas vulnerable to illegal logging activities.

---

## Study Workflow
1. Environmental data extraction using Google Earth Engine
2. Spatial preprocessing and variable preparation
3. MaxEnt ecological niche modeling
4. Model evaluation and suitability mapping
5. Spatial interpretation of risk distribution

---

## Tools & Software
- Google Earth Engine
- MaxEnt
- QGIS
- R (visualization & analysis)
